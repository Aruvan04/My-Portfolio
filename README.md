# Aruvan Mishra | Full Stack AI Portfolio

A high-performance, responsive portfolio website built with **Tailwind CSS**, **JavaScript (ES6+)**, and a custom **neural-network themed canvas engine**. This portfolio showcases technical proficiency, project architecture, and professional experience.

## 🚀 Features

*   **Neural Canvas Engine**: Custom-built interactive background featuring particle systems, signal propagation, and real-time 3D-cube rendering.
*   **Production-Ready UI**: Designed with a sleek "Dark Mode" aesthetic using a premium `Plus Jakarta Sans` and `Space Grotesk` typography stack.
*   **Optimized Performance**: Zero-dependency animation engine for smooth entrance transitions.
*   **Fully Responsive**: Fluid grid layouts that scale from mobile devices to ultrawide desktop monitors.

## 🛠 Tech Stack

*   **Markup/Style**: HTML5, Tailwind CSS
*   **Logic**: Vanilla JavaScript (ES6+)
*   **Visuals**: Custom HTML5 Canvas API
*   **Deployment**: Hosted/Deployed via Netlify

## 📁 Setup Instructions

To run this project locally:

1.  **Clone the repository**:
```bash
    git clone [https://github.com/Aruvan04/your-repo-name.git](https://github.com/Aruvan04/your-repo-name.git)
    ```
2.  **Open in Browser**:
    Simply open `index.html` in any modern web browser.
3.  **Image Configuration**: 
    The portfolio is currently set to use a minimalist initials-based avatar. If you wish to use your original professional portraits:
    *   Locate the `#profile-frame` ID in `index.html` to replace the initials `div` with your white shirt `<img>` tag.
    *   Locate the `About Section` grid to replace the feature box with your blue shirt `<img>` tag.
